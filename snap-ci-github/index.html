If you‚Äôre curious to see how I used gh-pages to deploy to specific branches and repositories, have a look at the deploy script for Fun Retrospectives and the one for this blog.

#!/bin/sh
LAST_COMMIT=$(git log -1 --pretty=format:'%h %an %s')
